Acerca de Mí

¡Hola! Soy Absalon, diseñador web apasionado por crear experiencias digitales atractivas, funcionales y centradas en el usuario. Con 19 años de experiencia en el campo, me especializo en diseño responsivo, UI/UX y desarrollo front-end, combinando creatividad y tecnología para dar vida a proyectos únicos.

Habilidades

Mi enfoque va más allá de la estética: busco entender las necesidades del cliente y su audiencia para ofrecer soluciones que no solo sean visualmente impactantes, sino también intuitivas y efectivas. Trabajo con herramientas como HTML5, CSS3, JavaScript y React, siempre actualizándome con las últimas tendencias del sector.

HTML

Es la base estructural de la web. Define el contenido y la semántica de una página mediante etiquetas modernas, permitiendo mejor accesibilidad y compatibilidad.

CSS3

Aquí empieza la magía, transforma el diseño web con estilos avanzados, animaciones, flexbox, grid y diseño responsivo. Gracias a CSS3, las páginas son visualmente atractivas y se adaptan a cualquier dispositivo.

Javascript

El lenguaje que da interactividad a la web. Desde validar formularios hasta crear aplicaciones dinámicas, JS es esencial para el desarrollo front-end (y back-end con Node.js).

React

Una librería de JavaScript creada por Facebook para construir interfaces de usuario eficientes y reutilizables. Su enfoque en componentes y el Virtual DOM optimiza el rendimiento, ideal para SPAs y aplicaciones escalables.
